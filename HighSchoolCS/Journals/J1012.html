1: Describe the general process for executing addition, reguardless of base?
The general rules for addition is to increase the addends by the given numbers. If given a number line, the number would increase in position by the amount that is added.
2: What special factors need to be considered when executing addition with octal numbers?
In the decimal system, there are digits 0 to 9, so we carry over when the digit is over 9. In octal addition, we carry over when the digits are over 7.
3: What special factors need to be considered when executing addition with hexidecimal numbers?
When executing addition with hexidecimal numbers, the digits need to be considiered. Because hexidecimal numbers include digits that have values over 10, the carrying over can only happen when the addition results in a number over 15.
4: What special factors need to be considered when executing addition with binary numbers?
When adding with binary numbers, the maximum digit that can be represented is a 2. Therefore any numbers that are even result in a 0 and a carry. Any odd numbers result in a carry and a 1 in the digit.
