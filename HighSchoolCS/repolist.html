The Allen Cyber Invitational 2023-2024 is a project dedicated to teaching students basic cybersecurity concepts in a fun and exciting manner. Through a responsive and interpretive environment, participants will tackle Linux-based puzzles covering various cyber security topics, such as cryptography, steganography, web exploitation, CLI commands, and binary exploitation. With approximately $9000 in funding, this project developed not just our programming skills but also our interpersonal skills. We hope that this project can continue to provide a beneficial learning experience for both competitors and developers for years to come.

Allen High School's Model Scheduler is a tool that Allen High School's computer science students develop each year. This tool assists Allen High School students when preparing to build their schedules for the next school year and by simulating the idea behind the Arena Scheduling system. As my first project, I learned how to work with React components and libraries and how to effectively create an efficient team workflow. Model Scheduler Repository

NASA Hunch Cyber Security is a forward looking project that envisions a system for a technologically safe lunar base. As space travel and space settlement gets closer and closer, the risks of having an unsafe network becomes more readily apparent. Along with the rapid development of space-related technology, quantum computers have also been improving quickly. Our project aims to tackle the dangers of space and quantum technology on information transfer between Earth and Space. NASA Hunch Cybersecurity Repository
