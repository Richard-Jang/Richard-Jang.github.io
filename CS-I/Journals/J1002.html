7.1: How does the "ls" command differ from the "tree" command?
The "ls" command only shows one path to the current directory. The tree command shows every directory and file under the home directory.
7.2: How do you choose to use one over the other?
The "ls" command should be used when the user only wants to see the path to the current directory. When looking for a certain directory and its location, use the "tree" command.
