1.1: Why do you think the Hello World program is typically the first program taught to students?
The Hello World program is a very simple input output program. It allows students to get used to the programming language, as well as learn a basic "print" command.
1.2: How might the Hello World program be different, depending on the programming language it is written in?
Different languages have different ways to output data. The Hello World program can be written differently depending on how the language outputs data.
