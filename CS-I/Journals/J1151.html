1.1: How does branching enable significantly more complex program behavior?

Branching allows for computer programs to execute different tasks depending on a conditional.


1.2: Why do you think the term consequent is used to refer to the statements that are executed when the Boolean test evaluates to true?

When boolean evaluates to true, consequent is executed because that is the main program, and the else is an alternative program that runs when boolean evaluates to false.


2.1: Why is the first Jump instruction conditional? What determines whether the jump will be executed or not?

The first jump is a conditional beause it compares the value and executes instructions based on the result. The operation of the jump command will determine whether the jump will be executed or not.


2.2: What is the purpose of the Jump instruction after the consequent?

The Jump instruction after the consequent acts as a nested if statement, it moves to the "afterAlternative" if run.


Why is there no need for a corresponding Jump instruction after the alternative?

The "afterAlternative" is already under "alternative" so there is no need to jump.
