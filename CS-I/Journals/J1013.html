1: Why is one of the symbols used for exclusive disjunction ≢ ?

The symbol for equivalence is ≡, and this means that the output is true iff both inputs have the same value. Exclusive Disjunctions means that the output is true if either the inputs are true but are not true, the opposite of equivalence, therefore resulting in the slash going through ≡ (≢).


2: Why is the formula for determining the number of rows in a truth table related only to the numberes of inputs, and why is the formula 2n?

The number of rows in a truth table is related only to the numbers of inputs because if each variable has two outcomes, true or false. Each row represent a possible outcome, so with n amount of inputs, there would be 2n amount of possibilities, resulting in 2n rows.
